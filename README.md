**HealthTrack & Fun** is a simple web application that combines **health tools and fun games** in one place.
Users can calculate their **BMI (Body Mass Index)** and also play a **Rock Paper Scissors game** against the computer.

The app also shows a **daily motivational quote** to keep users positive and motivated, these quotes are fetched by api and also on every loading changes the quotes.

---

# 🚀 Features

### 💬 Daily Motivation

* Shows a random motivational quote using an API.

### 🏥 BMI Calculator

* Enter height and weight
* Calculates **BMI (Body Mass Index)**
* Shows BMI category:
  * Underweight
  * Normal
  * Overweight
* Displays a progress bar and health message.

### 🎮 Rock Paper Scissors Game

* Play against the computer
* Scoreboard for **player vs computer**
* First to **5 points wins the match**
* Reset button to start a new game
* Sound effect when the player loses

# 💡 How It Works

### Motivational Quote

* Uses **Fetch API** to get quotes.
* Displays a random quote when the page loads.

### BMI Calculator

1. User enters height and weight
2. JavaScript calculates BMI

```
BMI = weight / (height × height)
```

3. Shows BMI value and health category, and also a message.

### Rock Paper Scissors

* User selects **rock, paper, or scissors**
* Computer randomly chooses
* Score updates after each round
* If computer wins it will give a sound fahhhhh
* First to **5 points wins the match**
